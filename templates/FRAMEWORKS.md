# Alternative Framework References

Each toolkit module uses specific established frameworks for grounding. These are not locked in -- you can swap them for alternatives that better fit your domain or preference.

## Module 1: Experience Design

**Primary**: Design Thinking (Brown, 2009), Jobs-to-Be-Done (Christensen et al., 2016)
**Secondary**: Service Blueprinting (Shostack, 1984), Backward Design (Wiggins and McTighe, 1998)

**Alternatives**:
- Kano Model (Kano et al., 1984) -- for categorizing experience attributes (must-be, one-dimensional, attractive)
- Customer Journey Mapping (Stickdorn et al., 2018) -- for temporal experience design
- Outcome-Driven Innovation (Ulwick, 2005) -- for systematic outcome identification
- SERVQUAL (Parasuraman et al., 1988) -- for service quality dimensions

## Module 2: Signal Architecture

**Primary**: Spectral Brand Theory 8 Dimensions (Zharnikov, 2026a)
**Secondary**: Service Design (Stickdorn et al., 2018)

**Alternatives**:
- Kapferer Brand Identity Prism (Kapferer, 2012) -- 6-facet model (physique, personality, culture, relationship, reflection, self-image)
- Keller CBBE Model (Keller, 1993) -- brand equity pyramid
- Aaker Brand Identity Model (Aaker, 1996) -- 12 identity elements across 4 perspectives
- Brand Touchpoint Analysis (Davis and Dunn, 2002) -- touchpoint-centric signal mapping

Note: SBT's 8 dimensions provide the most granular signal decomposition. Alternatives may work for specific analyses but lose dimensional resolution.

## Module 3: Process Contracts

**Primary**: SIPOC (Six Sigma), ISO 9001:2015 process approach
**Secondary**: BPM (Dumas et al., 2018), QFD (Akao, 1966)

**Alternatives**:
- BPMN 2.0 (OMG, 2011) -- for formal process modeling notation
- Value Stream Mapping (Rother and Shook, 1999) -- for lean process analysis
- IDEF0 (NIST, 1993) -- for functional modeling
- SCOR Model (APICS) -- for supply chain process classification

## Module 4: Procedure Specification

**Primary**: Toyota Standardized Work (Ohno, 1988; Liker, 2004)
**Secondary**: TWI Job Instruction, Cognitive Task Analysis (Crandall et al., 2006)

**Alternatives**:
- GOMS Model (Card, Moran, Newell, 1983) -- for cognitive procedure analysis
- Hierarchical Task Analysis (Annett and Duncan, 1967) -- for task decomposition
- SOP best practices per ISO 9001 -- for compliance-oriented procedures
- Work Instructions per ISO 10013 -- for QMS documentation

## Module 5: Input Specification

**Primary**: Bill of Materials (MRP/ERP), Supply Chain Management (Chopra and Meindl, 2019)
**Secondary**: ISO 22000 (food safety management)

**Alternatives**:
- ABC Analysis -- for inventory prioritization
- Vendor Managed Inventory (VMI) -- for supplier-managed input tracking
- Total Cost of Ownership (TCO) -- for comprehensive cost specification
- HACCP (Codex Alimentarius) -- for food industry input safety

## Module 6: Sourcing Requirements

**Primary**: ISO 9001:2015 clause 8.4 (Control of externally provided processes)
**Secondary**: Supplier Relationship Management (Monczka et al., 2015)

**Alternatives**:
- Kraljic Matrix (Kraljic, 1983) -- for supplier portfolio segmentation
- SCOR Model Source processes -- for standardized sourcing metrics
- Fair Trade / B Corp criteria -- for ethical sourcing frameworks
- ISO 20400 (Sustainable Procurement) -- for sustainability-focused sourcing

## Module 7: Signal Mapping

**Primary**: SBT Signal Decomposition (Zharnikov, 2026a)
**Secondary**: Service-Dominant Logic (Vargo and Lusch, 2004)

**Alternatives**:
- Brand Audit frameworks -- for mapping brand elements to operational sources
- Customer Value Proposition Canvas (Osterwalder et al., 2014) -- for value-signal alignment
- Touchpoint Matrix -- for mapping operational activities to customer-facing signals

## Module 8: Gap Analysis

**Primary**: TDD (Beck, 2003), Lean Muda Analysis (Ohno, 1988)
**Secondary**: Value Stream Mapping (Rother and Shook, 1999), FMEA (IEC 60812)

**Alternatives**:
- SWOT Analysis -- for strategic gap identification
- Maturity Model Assessment (CMMI) -- for process maturity gaps
- Balanced Scorecard (Kaplan and Norton, 1996) -- for multi-perspective validation
- ISO 19011 (Audit guidelines) -- for systematic audit methodology
